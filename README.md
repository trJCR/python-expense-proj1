# Expense Tracker

A command-line Python application for tracking and managing personal expenses.

## Features
- Add, view, and delete expenses
- Sort by category, amount, or name
- Export to JSON
- Input validation and error handling

## Technologies
- Python 3.x
- CSV for data storage
- JSON for exports

## How to Use
1. Run: `python main.py`
2. Select menu options 1-8
3. Follow prompts

## Project Structure
- main.py - Menu and UI
- expenses.py - Core logic (sorting, adding)
- file_handler.py - CSV I/O
- data/expenses.csv - Data storage
